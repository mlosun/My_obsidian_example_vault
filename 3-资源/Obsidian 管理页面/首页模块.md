#### 最近的 Fleeting Notes
```dataview
table without id 
	file.link as 日期,
	dateformat(created, "ccc") as 星期, 
	L.text as 闪念笔记, 
	L.link as 链接
from "0-周期笔记"
where type = "Daily"
flatten file.lists as L
where
	!L.parent and
	meta(L.section).subpath = "Fleeting Notes"
sort file.name desc,L.text desc
```
